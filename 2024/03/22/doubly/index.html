倍增

倍增是一种常见的算法思路，常常与动态规划配合使用。

倍增可以通过一系列2的幂，快速减小搜索空间，获得最终答案。接下来我们将通过一些题目说明倍增的使用场景。

区间最值问题（RMQ）

洛谷-ST表

题目描述

给定一个长度为 N 的数列(数列下标从1开始)，和M 次询问，求出每一次询问的区间[l,r]内数字的最大值。

思路

这一类型的题目也可以使用线段树/树状数组来做，但是也可以使用倍增结合动态规划完成。

首先区间内最大值存在这样的性质：

区间[l,r]的最大值等于区间 [l,k2]的最大值和[k1,r])的最大值中较大的那个，其中l <= k1 <= k2 <= r。也就是说，求解区间[l,r]的最大值可以转为求解两个可重叠的子区间的最大值。

定义f[i][j]为区间[i,i + 2^j]上的最大值,动态转移方程为f[i][j] = max(f[i][j - 1], f[i + 2 ^ (j - 1)][j - 1]，我们可以在\(\Theta(n\log n)\)时间内完成预处理。代码如下：

加速搜索过程

466. 统计重复个数

思路

题目希望在str1 = [s1, n1]中找到尽可能多的str2 = [s2, n2]，我们可以先找s2出现的次数x，x / n2就是最终的答案m。

最朴素的想法就是遍历整个str1，但是会超时，我们需要想办法加速遍历的过程。

假设str1中一共出现了x次s2，显然x可以通过二进制表示为\((a_n...a_1a_0)_2\)，，为了加速搜索过程，我们可以参照倍增的思想，从高位到低位搜索，依次确认\(a_n\)，\(a_{n-1}\)，...，\(a_0\)的值，最终在\(\Theta(\log m)\)的时间内得到x。 

使用动态规划:

  • 令dp[i][k]代表从s1[i]开始匹配\(2^k\)个s2需要多少个字符;

  • 动态转移方程：dp[i][k] = dp[i][k - 1] + dp[(i + dp[i][k - 1]) % l1][k - 1];
