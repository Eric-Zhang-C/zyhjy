6.小球碰撞

问题描述

数轴上有 \(N\) 个小球，小球可看作质点，第 \(i\) 个小球初始时位于 \(p_i\)，且每秒以 \(v_i\) 的速度运动，当 \(v_i > 0\) 时表示小球以每秒 \(|v_i|\) 的速度向右运动，当 \(v_i < 0\) 时表示小球以每秒 \(|v_i|\) 的速度向左运动，否则表示小球静止。

初始时你可以选择 \(K\) 个小球从数轴上移去，此时记作时间 \(T=0\)，你需要找到最大的时间 \(T'\)，使得满足一种移除 \(K\) 个小球的方案，在 \(T\in[0,T']\) 内均无小球发生碰撞，答案下取整保留为整数，特别地，如果 \(T'=+∞\)，输出 \(-1\)。

评测数据规模

对于所有测评数据，\(1 \leq K < N \leq 10^5,\forall 1 \leq i < j \leq N,p_i ≠ p_j,-10^9 \leq p_i,v_i \leq 10^9\)。

思路

首先思考，什么情况下小球不会碰撞？

小球的运动方向是固定的，初始位置也是确定的，因此两个小球最多相撞一次。考虑两个小球\(i,j\)，

  • 假设在\(T=0\)时\(p_i < p_j\)；
  • 在经过时间\(T\)后，
    • 如果\(p_i + (v_i \times T) < p_j + (v_j \times T)\)，即小球的相对位置没有发生变化，则说明这两个小球没有碰撞；
    • 否则两个小球相撞。

因此，\(T\)时刻不相撞小球的最大数目就等于满足小球相对位置关系与\(T=0\)时刻相同的子数组的最大长度。

对\(T=0\)时刻的小球按照位置由小到大排列,得到\(\{ p_1,..., p_i,...p_n \}\),

\(T\)时刻小球的位置是\(\{p_1 + (v_1 \times T),..., p_i + (v_i \times T),...p_n + (v_n \times T)\}\)。不相撞小球的最大数目就等于最长递增子数组的长度，记为\(L\)。

如果\(N - L \leq K\)，则我们可以通过移除 \(K\) 个小球，使得在 \([0,T]\) 内均无小球发生碰撞。

接下来使用二分查找，找到满足上述要求的最大时间 \(T'\)。

考虑到数据范围，\(-10^9 \leq p_i,v_i \leq 10^9\)，初始时两个小球可能处于数轴的两端，他们碰撞需要的最长时间为\(2 \times 10^9\)（相对速度为\(1\)的情况下）。所以二分查找的范围为\([0, 2 \times 10^9 + 1]\)，如果达到上界都没碰撞，返回-1。

小球数目\(N \leq 10^5\)，那么在计算最长递增子数组时，基于DP的复杂度O(N)的方法的会超时，需要使用复杂度为O(N*log N)的【贪心+二分查找】的方法。
