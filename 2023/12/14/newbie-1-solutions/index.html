1209 第 1 场 小白入门赛 题解

1. 蘑菇炸弹

问题描述

题莫种下了 \(n\) 个蘑菇炸弹，其中第 \(i(i\in[1,n])\) 个蘑菇的爆炸能力为 \(a_i\)。为此，瓦罗兰大陆著名的拆弹专家及格撕，受命来拆除这 \(n\) 个蘑菇炸弹，但他时间有限，只愿拆除那些高爆蘑菇炸弹。

若一个蘑菇炸弹满足以下条件则被称为高爆蘑菇炸弹。

  • \(2 \leq i \leq n-1\)。

  • \(a_i \ge a_{i-1}+a_{i+1}\)

请你帮及格撕求出高爆蘑菇炸弹的数量。

思路

\(a_i \ge a_{i-1}+a_{i+1}\) 等价于\(a_i - a_{i-1} \ge a_{i+1}\)，从第三个元素开始，判断其是否小于前两个元素之差即可。

2. 构造数字

问题描述

现在迪迦奥特曼有一个十进制下的 \(N\) 位正整数，但他忘记这个数的具体数值，只记得每一位的数码之和不超过 \(M\)，他想知道在所有可能中，这个数最大可能是多少。

即用 \(f(x)\) 表示十进制下 \(x\) 每位数码之和，例如 \(f(114)=1+1+4=6\)，给定 \(N,M\)，求最大的 \(N\) 位十进制数 \(x\)，满足 \(f(x) \leq M\)。

数据规模

对于所有测评数据，\(1 \leq N,M \leq 10^6\)。

思路

题目让我们将\(M\)分配给十进制数 \(x\)的 \(N\) 个位，显然：

  1. 每个位最大是9，最小是0。
  2. 高位越大越好。

因此采用贪心思路，从高位到低位进行数字分配，并且使得高位尽可能大，得到的结果就是最大值。数据规模不大，int足够了。

3. 小蓝的金牌梦

问题描述

最近，小蓝刚学习了经典问题中的最大子数组和问题。最大子数组和问题是 ACM 竞赛选手入门必学的一个问题，它要求在给定数组 \(a\) 中选取一段连续子数组，使得该子数组的元素和达到最大值。

当然小蓝对自己的要求比较高，恰巧他刚学习了质数的知识，他想知道如果选取的子数组长度必须为质数的情况下，最大的子数组和能是多少呢？

输入格式

第一行输入一个正整数 \(n\) 表示数组长度。第二行输入 \(n\) 个整数 \(a_1,a_2,a_3, \cdots,a_n\)。

输出格式

输出一个整数，表示长度为质数的最大子数组和。

评测数据范围

\(2 \leq n \leq 10^5\)，\(-10^5 \leq a_i \leq 10^5\)。

思路

先看数据范围，因为数组长度\(2 \leq n \leq 10^5\)，并且$ 10^5\(以内的质数的个数\)x ,000$，考虑暴力解答。

首先筛选出$ 10^5$以内的质数，筛质数我们一般使用埃氏筛法，时间复杂度是 O(n log log n)。然后对数组进行预处理，获得前缀和，之后比较每段长度为质数的区间和，找出最大值。

4. 合并石子加强版

问题描述

现在有 \(n\) 堆石子围成一个环，第 \(i\) 堆有 \(a_i\) 个石子，每次迪迦奥特曼可以选择相邻的 \(2\) 堆石子，将这两堆石子合并，合并后的石子个数为两堆石子个数之和，合并的代价为两堆石子个数之积，迪迦奥特曼想知道将这 \(n\) 堆石子合并成 \(1\) 堆的最小代价。

以下给出两堆石子相邻的解释：假设现在有 \(n\) 堆石子 \((n \geq 2)\) ，石子两堆石子 \(i,j\) 是相邻的，当且仅当 \(|i - j|=1\) 或 \(|i-j|=n-1\)。

输入格式

第一行包含 \(1\) 个正整数 \(n\)。

第二行有 \(n\) 个正整数，第 \(i\) 个表示权值 \(a_i\)。

评测数据范围

\(1 \leq n \leq 3 \times 10^4\)，\(1 \leq a_i \leq 2 \times 10^5\)。

思路

考虑第 \(i\)堆石子个数为 \(a_i\)，每次第 \(i\)堆（或包含第 \(i\)堆的中间状态）与第 \(j\)堆合并时，第 \(i\)堆对总代价的贡献是\(a_i \times a_j\)，那么在合并完成之后，第 \(i\)堆对总代价的贡献是\(a_i \times \sum_{\substack{j=1 \\ j \neq i}}^{n} a_j\)。

也就是无论以什么顺序合并，每堆石子对总代价的贡献都是固定的，因此总代价是定值。

接下来按照顺序合并所有石子堆即可。需要注意的是，结果超出了long long的范围，但是在unsigned long long之内。也可以考虑使用自己定义的int_128来保证运算结果不会溢出。

5. 简单的LIS问题

问题描述

最长上升子序列（Longest Increasing Subsequence），简称 LIS，指的是一个序列中最长的一个子序列，满足其单调递增。例如，对于一个序列 \(a_1, a_2, a_3, a_4,...,a_i,...,a_n\)，存在一个序列 \(b_1, b_2,..., b_m\)，满足：

  1. \(1 \le b_1 \lt b_2 \lt ... \lt b_m \le n\)。
  2. \(a_{b_1} \lt a_{b_2} \lt ... \lt a_{b_m}\)。

那么我们称 \(\lbrace a_{b_1} , a_{b_2} , ... , a_{b_m} \rbrace\) 为 \(\lbrace a_i \rbrace\) 的一个上升子序列。

最长的上升子序列就是最长（元素数量最多）的一个上升子序列。

小蓝学到了这个知识点，并且学会一些用来计算 LIS 的常见算法。

于是小蓝向邻居小桥炫耀他学到的新知识，但是小桥显然不会让小蓝如此得意，于是他提出了一个新的问题：

给定一个长度为 \(n\) 的序列 \(\lbrace a_i \rbrace\)，小蓝可以选择其中的一个位置，修改为 \(0 \sim 10^{100}\) 中的任何一个整数。具体来说，小蓝可以选择一个位置 \(p(p \in [1, n])\)，将 \(a_p\) 重新赋值为 \(0 \sim 10^{100}\) 中的任意一个整数。请问修改过后，最长上升子序列的长度是多少？

输入格式

第一行输入一个整数 \(n\)。

第二行输入 \(n\) 个整数 \(a_1, a_2, ..., a_i, ... ,a_n\)，用来表示序列 \(a\) 的每个值。

评测数据范围

\(1 \le n \le 5 \times 10^3, 0 \le a_i \le 10^9\)。

思路

不难证明，修改之后，最多可以使得最长上升子序列的长度+1。

分类讨论：

  1. 如果原先最长上升子序列的起始下标$start > 1 \(,那么可以将下标小于\)start$的某个值修改为0；此时，得到的结果长度+1。
  2. 同样的，如果原先最长上升子序列的终止下标$end < n \(,那么可以将下标大于\)end$的某个值修改为\(10^{10}\)；此时，得到的结果长度+1。
  3. 如果原先最长上升子序列中，存在相邻的两个元素\(\lbrace a_{b_i} , a_{b_j} \rbrace\)满足\(a_{b_i} + 1 < a_{b_j}\) 并且 \(b_i + 1 < b_j\)，那么可以将下标为\(b_i + 1\)的元素值修改为\(a_{b_i} + 1\)，此时，得到的结果长度+1。
  4. 若不满足上述三种情况，修改之后，最长上升子序列的长度不变。

6. 期望次数

问题描述

给定长度为 \(N\) 的序列 \({p}\)，初始时有一个正整数 \(x=1\)，不断进行如下操作：

  1. 若 \(x \geq M\) ，则操作结束
  2. 否则以 \(\frac{p_i}{\sum_{j=1}^N p_i}\) 的概率，使得 \(x\) 变为 \(x \times i\)

请你求出期望进行的步骤 \(2\) 的次数，答案对 \(998244353\) 取模。

评测数据规模

对于所有测评数据，\(2 \leq N,M \leq 10^5，1 \leq p_i,\sum_{i=1}^N p_i < 998244353\)。

思路

这是一道典型的期望DP，假设\(dp[i]\)表示\(x = i\)时，期望进行的步骤 \(2\) 的次数, 令\(S = \sum_{i=1}^N p_i\)。

Base Case：当\(i \geq M\) 时，此时强制进行步骤一，操作结束，因此\(dp[i] = 0\).

接下来考虑一般情况：当\(x = i < M\) 时，\(x\)有\(\frac{p_i}{\sum_{j=1}^N p_i}\) 的概率变为 \(x \times i\)，由此我们得出状态转移方程：

\(dp[i] = \mathbf{1} + \sum_{j=1}^N{\frac{p_j \times dp[i \times j]}{S}}\).

注意这里的+1，这是指经过一次步骤2之后，\(x\)可能按照相应概率变成\(i \times j\)，因此\(dp[i]\)的期望等于下一状态的平均期望+1。

注意到两边都有\(dp[i]\)移项之后得到递推公式：

\(dp[i] = \frac{S + \sum_{\substack{j=2}}^N{p_j \times dp[i \times j]}}{S - p_1}\).

其他：对有理分数取模需要用到逆元的概念。
