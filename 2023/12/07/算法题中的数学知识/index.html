组合数

乘法逆元

\((a/b)\%c=(a * b^{-1}) \% c\)

  • 什么是乘法逆元?

    • 求 (A/B) % MOD
    • 考虑余数定理, 希望转化成 (A * X) % MOD 的形式
    • X就是乘法逆元

  • 怎么求乘法逆元?

    1. 如果 MOD 是素数, 那么使用费马小定理
    2. 如果 MOD 不是素数, 且 \(gcd(X,MOD) = 1\), 那么, 可以使用扩展的欧几里得

费马小定理

  • 如果 MOD 是素数, 那么 B 对 MOD 的逆元是 \(inv(B, MOD) = B^{MOD - 2}\)
  • 怎么求 \(B^{MOD - 2}\) 用快速幂, 最后 \((A/B) \% MOD = [(A \% MOD) * (inv(B, MOD) \% MOD)] \% MOD\)
  • 算法: 大素数判断 + 快速幂

#include <vector>

using namespace std;

typedef long long ll;

ll fastpow(const ll& x, const ll& y, const ll& m) {
    ll a = x % m, b = y;
    ll ans = 1;
    while (b != 0) {
        if ((b & 1) == 1) {
            ans *= a;
            ans %= m;
        }
        a *= a;
        a %= m;
        b >>= 1;
    }
    return ans;
}

ll extgcd(const ll& a, const ll& b, ll& x, ll& y) {
    if (b == 0) {

}
